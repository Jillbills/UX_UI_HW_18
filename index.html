Edúcate Ya Case Study

Edúcate Ya

Non-Profit Responsive Website Redesign

Edúcate Ya is a non-profit organization that helps educate Latinos to achieve better jobs and access to healthcare. We redesigned the non-profit organization’s website to be responsive and visually engaging to improve usability, navigation, and user engagement.







Project Overview

The Problem

Edúcate Ya hasn’t been reaching it’s targeted audience. Their homepage averaged an 85% bounce rate in Oct to Nov 2021, which is nearly twice as high as a typical website bounce rate.

The Solution

By redesigning Edúcate Ya’s website to be more visually engaging, we can help improve usability, navigation, and user engagement.
